# GymFriend 💪🍱

**GymFriend** はジムに通う人のためのAI食事分析アシスタントです。
（糖尿病患者向けの「DiabetesFriend」のジム版。回答はすべて**日本語**です。）

食べ物を撮影・入力すると、Google Gemini が **カロリー・タンパク質・PFCバランス** を
推定し、筋肉づくり・体脂肪管理の観点で評価・アドバイスします。

## 🌟 主な機能

1. **📷 画像分析** — 食事の写真を撮影/選択すると AI が栄養を分析・評価（ストリーミング表示）
2. **📝 テキスト分析** — 食べ物の名前を入力するだけで同様の分析
3. **🍱 食事プラン生成** — ジム向けの1日の食事プランを自動生成
4. **💬 追問（フォローアップ）** — 各記録に対してチャットで追加質問が可能
5. **✏️ プロンプト編集** — アプリ内でAIへの指示文を自由に微調整できる
6. **🔑 APIキー入力** — サーバー不要。Gemini APIキーをアプリ内に入力するだけ
7. **🔒 ローカル保存** — 記録・画像・設定はすべて端末内(Hive)に保存。オフラインで閲覧可
8. **🔠 文字サイズ調整** — 詳細画面で文字サイズを変更可能

## 🛠️ 技術構成

* **フロントエンド**: Flutter (Dart)
* **AIモデル**: Google Gemini 3.1 Pro（モデルIDは設定で変更可能）
* **API呼び出し**: `generativelanguage.googleapis.com` を**直接**呼び出し（プロキシ/サーバーなし）
* **ローカル保存**: Hive（記録・設定・編集したプロンプト）
* **画像**: image_picker / gal

> **DiabetesFriend からの主な変更点**
> - サーバー/プロキシ（Vercel）と MongoDB を**廃止** → Gemini を直接呼び出し、データは端末ローカルのみ
> - 「問病問診」「自由聊天（フリーチャット）」を**削除**
> - プロンプトを**アプリ内で編集可能**に（設定画面）
> - APIキーを**アプリ内で入力**
> - 出力言語を**日本語**に、フィットネス向けの評価（最高/良い/普通/注意/避けるべき）とサマリー（カロリー・タンパク質）に変更

## 🚀 セットアップ

```bash
flutter pub get
flutter run
```

### 初回設定
1. アプリ右上の ⚙️ **設定** を開く
2. [Google AI Studio](https://aistudio.google.com/app/apikey) で取得した **Gemini APIキー** を入力
3. 必要ならモデルID（既定: `gemini-3.1-pro`）やプロンプトを調整
4. **保存** を押す

> APIキー・記録・編集したプロンプトはすべて端末内にのみ保存されます。

## 📁 プロジェクト構成

```
lib/
├── main.dart                    # アプリ本体（一覧・詳細・プロバイダー）
├── prompts.dart                 # 編集可能プロンプトのデフォルト値
├── models/
│   ├── food_entry.dart          # 記録モデル（Hive）
│   └── food_entry.g.dart        # Hive アダプター
├── services/
│   ├── gemini_service.dart      # Gemini API 直接呼び出し（ストリーミング）
│   └── settings_service.dart    # APIキー・モデル・プロンプトの保存/管理
└── screens/
    └── settings_screen.dart     # 設定画面（APIキー・プロンプト編集）
```
